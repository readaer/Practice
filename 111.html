HTML 문법 - 태그

태그란?

태그(Tag)란 HTML 문서를 구성하는 기본 단위로서, HTML의 문법을 지시하는 기능을 한다.
열린 태그와 닫힌 태그로 구분하여 부르기도 한다. 태그는 중첩하여 사용이 가능하다.

속성이란?

속성(attribute)란 태그 안에서 사용되는 좀 더 구체적인 명령어 체계를 말한다.

태그 종류

기본 형식 태그

  1. !doctype: 문서의 버전에 관한 정보를 나타냄. 예를 들어 [!doctype html]이면 본 문서가 HTML5라는 것을 명시.
  2. html: html 파일임을 정의. html 태그는 head 태그와 body 태그로 이루어짐.
  3. head: 메타 데이터의 집합으로 웹 페이지에는 직접적으로 보이지 않는 정보임. head 태그는 title, meta, link, style, script 태그로 구성됨.
  4. meta: 문서에 대한 정보를 넣어주는 역할. 문자 인코딩 정보, 작성자 정보, 키워드, 설명 등. [meta 속성="속성값"]으로 스스로 닫는 태그.
  5. title: 페이지의 이름을 나타냄.
  6. body: 본문의 내용을 담는 공간.

그 외의 태그

  1. a: 하이퍼링크를 정의함. 함께 자주 쓰이는 속성은 링크를 나타내는 href.
    2. 다른 속성:
    1. target: 새창 or 새 탭에서 링크를 열 때 사용 - _blank, _self, _parent, _top
    2. title: 링크의 툴팁을 표시(커서 올렸을 때 나오는 설명)
    3. id: 같은 페이지 안에서 이동할 때 사용. 이동하고 싶은 위치마다 id 속성 이용하여 앵커를 만들고 다시 href로 연결.
  2. img: 이미지를 정의함. 함께 자주 쓰이는 속성은 src(경로)와 alt(이미지를 설명하는 텍스트), height와 width가 있음.
    3. 다른 속성:
    1. title: 툴팁을 표시
    2. usemap: 이미지 맵(하나의 이미지에 여러 개의 링크를 만드는 것)
  3. p: 단락을 정의함. p 앞뒤에 자동으로 여백이 생김.
  4. br: 줄바꿈을 정의함.
  5. li, ul, ol: 목록을 정의함. ul은 정렬되지 않은 목록, ol은 정렬된 목록.
    6. ol 태그의 속성:
    1. type: 순서 표시 형태 - 1, a, A, i, I
    2. start: 중간부터 시작해야 할 때
    3. reversed: 역순으로 할 때
  6. hn: h1~h6까지 본문 내용임을 중요도 순대로 정의함.
  7. strong: 중요한 텍스트를 정의함. bold 태그와 마찬가지로 굵게 표시됨.
  8. b: 글자를 굵게 표시함. 다만 strong 태그처럼 중요하다는 의미를 지니지는 않음.
  9. i: 글자를 기울임체로 표시함.
  10. em: 글자를 기울임체로 표시하면서, 실질적인 강조의 의미를 지님.
  11. div: 콘텐츠를 목적에 따라 묶을 때 사용함. 줄바꿈O.
  12. span: 콘텐츠를 목적에 따라 묶을 때 사용함. 줄바꿈X.
  13. script: 실행 가능한 코드를 삽입할 때 사용함. 주로 Javascript 코드를 넣을 때 사용.
  14. link: 외부 파일을 연결할 때 사용함.
  15. style: 스타일 정보를 정의할 때 사용하는 태그. head 태그 사이에서 정의.
  16. nav: 같은 사이트 안의 페이지나 다른 사이트의 페이지로 연결하는 링크.
  17. iframe: 웹 페이지 안에 다른 웹 페이지를 삽입할 때 사용.
    width, height, name, seamless, src 등의 속성과 함께 쓰인다.
