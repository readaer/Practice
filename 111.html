HTML 문법 - 태그

태그란?

태그(Tag)란 HTML 문서를 구성하는 기본 단위로서, HTML의 문법을 지시하는 기능을 한다.
열린 태그와 닫힌 태그로 구분하여 부르기도 한다. 태그는 중첩하여 사용이 가능하다.

태그 종류

기본 태그

!doctype : 문서의 버전에 관한 정보를 나타냄. html : html 파일임을 정의. html 태그는 head 태그와 body 태그로 이루어짐. head : 메타 데이터의 집합으로 웹 페이지에는 직접적으로 보이지 않는 정보임. head 태그는 title, meta, link, style, script 태그로 구성됨. meta : meta 태그는 으로 스스로 닫는 태그. title : title 태그는 페이지를 대표하는 내용의 제목을 넣는 자리. body : 본문의 내용을 담는 공간.
